# Medium + Substack Publishing Playbook

## 1. Source assets
Start from your Jasper canvas:
- Final essay document
- Final publication pack
- Featured image file

## 2. Open the publishing destinations
- **Medium new story:** https://medium.com/new-story
- **Substack dashboard:** https://substack.com/dashboard

## 3. Create the Medium draft
Go to:
- https://medium.com/new-story

Then:
- paste in the final essay
- add the final headline
- add the subtitle if you’re using one
- insert the featured image at the top
- check paragraph spacing
- check headings
- check bold/italics
- re-add or verify links
- add tags before publishing

### Medium pre-publish checks
- headline is final
- featured image displays well
- opening paragraph is strong
- section breaks look clean
- tags are added
- mobile preview looks good

## 4. Create the Substack draft
Go to:
- https://substack.com/dashboard

Then:
- create a new post
- paste in the essay
- add the featured image
- add the Substack intro note
- add the closing CTA
- verify formatting in both web and email preview
- publish or schedule

### Substack pre-publish checks
- title is final
- intro note feels personal, not repetitive
- closing CTA is present
- image works at top of post
- email preview looks clean
- links work
- spacing is consistent

## 5. Image handling
Before uploading:
- use a final image around **2000 × 1125 px**
- confirm the crop works at thumbnail size
- make sure no edge details are essential
- confirm any incidental text in the image is not distracting

## 6. Copy to bring over
From your canvas, carry across:
- the main essay
- final headline
- final subtitle/dek
- Medium tags
- Substack intro note
- Substack CTA
- LinkedIn launch copy

## 7. Final QA pass
Before publishing on either platform, check:
- no typos in title/subtitle
- all links work
- formatting survived paste
- no duplicated paragraphs
- no broken section spacing
- image appears correctly
- ending lands cleanly

## 8. Publish
Suggested order:
1. publish on **Substack**
2. publish on **Medium**
3. publish the **LinkedIn launch post**
4. later, cross-link the companion piece **A Minimalist AI Tech Stack**

## 9. Post-publish tasks
After publishing:
- copy the live URLs into your notes
- add cross-links between essays
- update LinkedIn/social with the live link
- save the final headline, subtitle, and launch copy that performed best

## 10. Useful URLs
### Medium
- New story: https://medium.com/new-story
- Home: https://medium.com/

### Substack
- Dashboard: https://substack.com/dashboard
- Login: https://substack.com/sign-in

## 11. Reusable mini workflow
For any future essay:
1. finalize article in canvas
2. prepare publication pack
3. choose headline/dek
4. prep image
5. paste into Medium
6. paste into Substack
7. QA both
8. publish
9. launch on LinkedIn
10. archive links and learnings
