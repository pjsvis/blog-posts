# My 2026 AI-Assisted Coding Stack

AI coding tooling in 2026 suffers from excess rather than scarcity. The useful question is not "what’s new?" but "what actually reduces conceptual entropy at its layer?"

Here is the stack I use daily.

- **Zed** — The editor. Fast, native AI integration, excellent model routing.
- **herdr** — Terminal workspace manager purpose-built for agent workflows. Persistent panes, agent-aware sessions.
- **Pi** — The agent harness and governance layer. Loads the Edinburgh Protocol as persona.
- **Bun** — Runtime. Blazing cold-start performance for agent scripts.
- **Hono** — Lightweight web framework when an API surface is needed.
- **CLI Layer** (`td` + `just`) — Task management and command contracts.

Each tool has a clear responsibility. No overlap, minimal cognitive load. The agent knows where to go and what to use.

The stack is not sacred. It is the current local optimum. Examine your own friction points and replace one piece at a time.
