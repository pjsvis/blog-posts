# Silo

A folder on a disk that functions as a self-contained working environment for an AI agent.

Not just storage — a complete context. The silo contains:

- **Conventions** — how the project is organized, how work is done
- **Lexicon** — the shared vocabulary of the project (Stuff, Things, Mentation, etc.)
- **Identity** — project-specific operating rules (AGENTS.md)
- **Verbs** — the defined commands available in this context (justfile)
- **Registry** — a record of what exists and where (INDEX.jsonl files)
- **Artefacts** — briefs, debriefs, decisions, visitor's book entries

The agent enters the silo and orients. It reads the conventions. It knows where it is and what it can do. The data lives in the silo. The network has nothing to do with it.

This is a Palm Pilot for the AI era.
